*****Command to run test in this framework****
<behave_html_formatter Report>

<Using folder name>
behave Features -D environment=QA -D browser=Chrome --f html -o Reports/behave-report.html

<Using tag>
behave Features --tags=@Sanity -D environment=QA -D browser=Chrome -f html -o Reports/behave-report.html

<Using Feature file name>
behave Features/Registration.feature -D environment=QA -D browser=Chrome --f html -o Reports/behave-report.html



***********Framework setup steps*************

1. Install Pycharm
2. Install Intellibot plugin
3. Create Project structure (folders)
	Step definition folder must be labeled as "Steps" or "steps"
4. Check if PIP is installed
5. Add dependency libraries
	selenium
	behave
	behave-html-formatter
	openpyxl
	jsonpath
	selenium-page-factory
5. Add browser drivers in drivers folder
6. Create behave.ini file
7. Create Base.py file inside global folder
8. Create Environment.py file inside features folder
9. Add Utilities inside utilities folder (optional)
10. Add Test Data in testDatafolder
11. Add feature file in features folder
12. Add Step Definition file inside steps folder in features folder
13. Add methods file pages folder
